## Overview

The **RAG API** is an intelligent search solution designed to leverage documents (files, URLs, knowledge bases, procedures, guides) with modern AI models (LLMs).

It transforms raw documentation into a **knowledge base queryable in natural language**.

---

## Background

This solution is the direct result of an [internal RAG Benchmark](https://github.com/Kick-My-Bot/rag-benchmark), a comprehensive comparative study of different RAG approaches:

- Vector search
- Lexical search
- Hybrid search
- Reranking strategies
- Semantic chunking

It materializes the conclusions of this research by offering an optimized, robust architecture ready for production.
