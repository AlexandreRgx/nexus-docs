## Guidelines

### Before Using a Template

1. Check if similar content exists
2. Discuss informally if needed
3. Gather the necessary context

### During Writing

1. Fill in all sections (or mark N/A)
2. Be concise but complete
3. Include examples and diagrams
4. Link to existing documentation

### After Publication

1. Request appropriate reviews
2. Address feedback quickly
3. Keep content up to date

---

## Contributing a New Template

1. Open an issue describing the need
2. Write the template following existing patterns
3. Include usage guidelines
4. Submit a PR
